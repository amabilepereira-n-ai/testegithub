O que é HTML?

HTML (HyperText Markup Language) é a linguagem de marcação usada na maior parte dos sites e aplicativos disponíveis na internet. Ela é constituída por tags que demarcam os elementos que serão exibidos na página.

Dica: Pense no HTML como a estrutura de uma casa — define o que existe e onde fica. O CSS cuida da aparência e o JavaScript do comportamento.

Estrutura básica do HTML

Todo documento HTML válido precisa seguir essa estrutura mínima. O <!DOCTYPE html> avisa ao navegador que é HTML5.

<!DOCTYPE html> — obrigatório na primeira linha
<html lang="pt-br"> — define o idioma (importante para SEO e acessibilidade)
<head> — metadados invisíveis ao usuário
<body> — tudo que aparece na tela
Estrutura mínima — edite e execute

Hierarquia DOM

O HTML opera através de uma hierarquia de pais e filhos — o DOM (Document Object Model). Um elemento só faz sentido dentro do seu contêiner correto.

Regra de ouro: todo elemento aberto deve ser fechado na ordem correta. Elementos filhos ficam dentro dos elementos pais.
